## How to run this on your device

- Clone this repository
```
git clone https://github.com/kingdreamerr/Day14_Higher_Lower.git
```
- cd into the repo
```
cd Day14_Higher_Lower
```

- Paste the following in the terminal 
```
python3 main.py
```
